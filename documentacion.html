Bienvenido

Documentacion

InceneroarUI es un framework CSS construido con SASS bajo el patron 7-1. Esta guia recorre cada pieza del sistema con un ejemplo vivo a la izquierda y el HTML exacto que tienes que copiar a la derecha.

Mobile First. Todas las clases responsivas (col-md-*, d-lg-flex...) usan min-width: definen lo que pasa desde ese breakpoint hacia arriba.
01 · Empezar

Instalacion

InceneroarUI te ofrece dos formas de trabajar dependiendo de tu flujo de desarrollo. Descarga el framework directamente o vincula la hoja de estilos compilada.

↓ Descargar Framework (.zip)
01 Implementacion Rapida (CSS)

Si solo necesitas los estilos finales, utiliza el archivo pre-compilado en produccion: 

<link rel="stylesheet" href="css/main.css">

Ubicacion: Carpeta css/

02 Personalizacion (SASS)

Si quieres modificar variables y colores originales, usa los archivos fuente:

  • Carpeta: scss/
  • Requisito: Extensión Live Sass
  • Uso: Edita _variables.scss

Si trabajas con SASS directamente

Importa los modulos que necesites en tu archivo principal: 

// main.scss
@use 'abstracts/variables';
@use 'abstracts/mixins';
@use 'base/reset';
@use 'components/buttons';
@use 'components/cards';
// ... resto de modulos

Estructura del Framework

Carpeta Tipo de Archivo Proposito
css/ .css (Compilado) Implementacion Archivo optimizado para el navegador
scss/ .scss (Fuente) Desarrollo Codigo modular para personalizacion avanzada
Recomendacion. Para proyectos en produccion usa el CSS compilado. Para desarrollo y personalizacion profunda, trabaja con los archivos SCSS y compila cuando necesites.

Tarjetas

Utilidades

Clases utilitarias

Clases de un solo proposito con !important que se aplican directamente al markup. Ideales para ajustes rapidos sin escribir CSS personalizado.

Espaciado

m-* para margen, p-* para padding. Sufijos: top, bottom, start, end, x, y. Escala 0-5.

.p-1
.p-3
.p-5
<div class="p-1">Padding 1 (4px)</div>
<div class="p-3">Padding 3 (16px)</div>
<div class="p-5">Padding 5 (48px)</div>

<div class="mt-3">Margen superior</div>
<div class="mx-4">Margen horizontal</div>

Flexbox

Activa flex con .d-flex y combina con justify-content-*, align-items-*, flex-* y gap-*.

A B C
gap-3 gap-3 gap-3 
<div class="d-flex justify-content-between align-items-center">
  <span>A</span><span>B</span><span>C</span>
</div>

<div class="d-flex gap-3 justify-content-center">...</div>

Display responsivo

Muestra u oculta elementos segun el tamano de pantalla con d-{bp}-{value}.

Solo en movil (oculto desde md)
Solo en escritorio (visible desde md) 
<div class="d-block d-md-none">Solo en movil</div>
<div class="d-none d-md-block">Solo en escritorio</div>

Animaciones

Animaciones

Cada animacion es una clase modificadora opcional que puedes sumar al componente base. Pasa el cursor o haz clic para verlas en accion. Respetan prefers-reduced-motion.

Botones animados

<button class="btn btn--submit">Submit</button>
<button class="btn btn--login">Login</button>
<button class="btn btn--delete">Delete</button>
<button class="btn btn--cancel">Cancel</button>
<button class="btn btn--edit">Edit</button>
<button class="btn btn--back"><span>&lt;</span> Back</button>
<button class="btn btn--next">Next <span>&gt;</span></button>
<button class="btn btn--download">Download <span>&darr;</span></button>
<button class="btn btn--refresh"><span>&#10227;</span> Refresh</button>

Tarjetas animadas

card--intimidate: se eleva con sombra roja al hover. card--selectable: cursor de seleccion. card--entrance: aparece en cascada al cargar.

Intimidate

Pasa el cursor para ver el efecto de elevacion con sombra de marca.

Selectable

Toda la tarjeta es interactiva. Haz clic para sentir el efecto de hundimiento.

Entrance

Aparece con un rebote al cargar la pagina, con cascada automatica.

<div class="card card--intimidate">...</div>
<div class="card card--selectable">...</div>
<div class="card card--entrance">...</div>

Formularios animados

Los .form-control ya incluyen elevacion al focus. .is-invalid aplica un temblor de retroceso. 

<input class="form-control">             <!-- focus eleva -->
<input class="form-control is-invalid">  <!-- vibra -->
<input class="form-control is-valid">    <!-- brillo verde -->
<input type="checkbox">                  <!-- burst al marcar -->
