Construye interfaces con fuerza y precision.

InceneroarUI es un framework CSS Mobile First inspirado en la intensidad del combate. Sistema de rejilla de 12 columnas, utilidades atomicas, componentes accesibles y animaciones cuidadas — todo escrito con SASS y arquitectura 7-1.

Por que InceneroarUI

Tres pilares, cero compromisos

Pensado para escalar desde un landing hasta un dashboard completo.

01

Arquitectura modular

Patron 7-1 con SASS: variables, mixins, base, layout, componentes y utilidades. Cada pieza vive en su archivo y se importa con @use.

abstracts · base · components · utilities
02

Sistema de utilidades

Clases atomicas para spacing, flexbox, tipografia, colores, bordes y sombras. Maquetas mas rapido sin escribir CSS personalizado.

m-* · p-* · d-flex · gap-* · rounded
03

Animaciones fluidas

Mas de 15 efectos listos para botones, tarjetas, alertas, formularios, tablas, badges y navbar. Curvas de aceleracion cuidadas, sin librerias.

btn--submit · card--intimidate · alert--victory

Mobile First de verdad, no como adorno.

Cada media query usa min-width: el sitio se construye desde el movil hacia arriba. Cuatro breakpoints (sm, md, lg, xl) cubren desde un telefono en vertical hasta una pantalla 4K.
